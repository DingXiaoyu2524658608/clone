#### 小组2的日常

1. 每一个任务周期（视难度，一到两周内），确定两个算法主题，难度分别在NOIP以上和以下。组员根据自身情况，选择一个主题，学习并完成该主题下题目。周期末时，举行线下交流会，讨论近期所学的内容、所做的题目等。

2. 每3-10天给小组1的同学布置新的任务，并提供你认为比较好的教程。

   任务的难度、持续时长自行把握。

   每一任务结束后，需要进行任务的完成情况统计。
